찾아오시는 길

충청북도광역치매센터로 찾아오시는 길을 안내해 드립니다.

주소

도로명 주소 : 충북 청주시 서원구 1순환로 767 지오빌딩 3층 우 28666

번지 주소 : 충청북도 청주시 서원구 개신동 50-5 3층 우 28666

연락처

043 269-6891~3

교통정보

택시 요금

요금 약 4,000 ~ 5,000원정도 (4.5km정도)

버스 이용 시

고속버스 이용시 (출발지 → 목적지) 서울,동서울,상봉,대구,부산,광주시고속터미널 → 청주고속버스터미널 → 823버스 탑승 →충북대학교병원정류장 하차
시내버스 기본요금(신탄진,부강,척산,미평행)

자가용 이용 시

(출발지 → 목적지) 각도시 → 충북 청주시 서원구 1순환로 767 지오빌딩3층 (가능하면 청주 I.C를 이용하시기 바랍니다.)

청원 I.C 청원I.C(좌회전) → 청주방향(직진) →미평4거리(좌회전:10.7km) → 충대병원5거리
서청주I.C 서청주I.C(중부고속도로,좌회전) → 봉명사거리(우회전;5.3km) → 사창4거리(직진:6.3km) → 충대병원5거리
청주I.C 청주I.C(경부고속도로) → 청주.보은.속리산방향 → 공단5거리(직진:6.8km) → 사창4거리(우회전:8km) → 충대병원5거리 충대병원 5거리에서 하나로 약국방면 좌회전 후 첫번째 골목 좌회전 미니스톱편의점 맞은편 우성문구 주차장에 주차.
